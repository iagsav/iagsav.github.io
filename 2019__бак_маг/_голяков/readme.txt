Необходимые зависимости для установки сервера:
1. Установленный Nodejs версии не ниже 11.3.0 
https://nodejs.org/en/
2. Установленный Npm версии не ниже 6.4.1, обычно идет в комплекте с установочными файлами nodejs 
https://www.npmjs.com/get-npm

Процесс установки:
Открыть папку проекта в командной строке
> npm i
> npm start

Сервер будет доступен по адресу https://localhost:4000
Дополнительных настроек не требуется


Чтобы установить обратный ssh-туннель необходимо на роботе ввести команду pi$ ssh -fN -R 11000:localhost:22 relayserver_user@1.1.1.1
relayserver_user@1.1.1.1 - это имя пользователя и адрес сервера
На сервере проверить, что адрес 127.0.0.1:11000 привязан к sshd командой netstat -nap | grep 11000

Дальше необходимо обменяться ssh ключами, для отправки команд с сервера на робота без пароля.
pi$ ssh-keygen
pi$ ssh-copy-id relayserver_user@1.1.1.1
